如何更新網站

若協同者有更新 進入.git裡的config 

則將【remote "origin"】裡的學號更改為更新者學號 之後按save

進入黑窗 先進小組倉儲(stage2-bg3)之後打git pull

pull完後 git add .

            git commit -m "add"

            git push bg3(看當初設定的名稱)
